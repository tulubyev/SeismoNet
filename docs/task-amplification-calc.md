# Расчёт усиления и отклика конструкций

## What & Why
Реализовать инженерно-расчётный модуль: метод тонкослоистых сред (МТСМ) для 1D расчёта усиления сейсмического сигнала в грунтовом разрезе, расчёт спектров ответа конструкций для типичных байкальских землетрясений и прогноз резонансных условий "грунт–здание".

## Done looks like
- Страница "Расчёты" с тремя вкладками: МТСМ, Спектр ответа, Анализ резонанса
- **МТСМ**: пользователь выбирает грунтовый профиль из БД (или вводит вручную), задаёт параметры (базовый полупространство Vs, глубина залегания скальных пород), нажимает "Рассчитать" → система вычисляет передаточную функцию методом Хаскелла–Томсона и отображает график усиления A(f) в диапазоне 0.1–20 Гц
- **Спектр ответа**: выбор сейсмического воздействия из каталога (типичное байкальское Мw 5.0 на расстоянии 100 км; Мw 6.0 на 200 км), выбор грунтовых условий → расчёт спектра ответа Sa(T) согласно СП 14.13330, графическое отображение
- **Анализ резонанса**: сопоставление собственной частоты здания (оценка по формуле T≈0.1·N этажей) с пиком H/V и резонансной частотой МТСМ → цветовая индикация уровня риска резонанса (зелёный/жёлтый/красный)
- Результаты расчётов сохраняются в БД с датой и параметрами
- Кнопка экспорта результатов в PDF/CSV

## Out of scope
- Полноценный МКЭ расчёт (требует специализированного ПО, возможна интеграция позже)
- Нелинейные модели грунта
- Детальный расчёт повреждений конструкций

## Steps
1. **Таблица расчётов в схеме** — `calculations` (тип расчёта, id профиля грунта, id объекта, входные параметры JSON, результаты JSON, дата, автор).
2. **Алгоритм МТСМ** — Реализовать матричный метод Хаскелла–Томсона на TypeScript/JavaScript: входные данные — массив слоёв [толщина, Vs, Vp, плотность], выходные — передаточная функция H(f). Выполняется в браузере (Web Worker для производительности).
3. **API расчётов** — Endpoint для сохранения и получения результатов расчётов.
4. **UI МТСМ** — Форма выбора профиля + ввод параметров базового полупространства; после расчёта — интерактивный график A(f) на D3 с подсветкой пиков усиления.
5. **Спектры ответа** — Каталог из 3–5 типовых сейсмических воздействий (акселерограммы или параметрические описания) для зоны Прибайкалья; расчёт Sa(T) и Sd(T); отображение графика.
6. **Анализ резонанса** — Форма: выбрать здание из БД объектов + выбрать грунтовый профиль → автоматическое сравнение частот с выводом вердикта и рекомендаций.
7. **Сохранение и экспорт** — Хранение результатов в БД, экспорт в CSV.
