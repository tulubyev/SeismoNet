# База данных грунтов и объектов Иркутска

## What & Why
Создать раздел "Грунты" с базой данных геологических условий в точках наблюдения: типы грунтов по СП 14.13330.2018, скоростные профили (Vs, Vp), данные буровых скважин. Это исходные данные для расчётов усиления сигнала. Включает интерактивную карту Иркутска с грунтовыми условиями.

## Done looks like
- Страница "Грунты" с двухпанельным интерфейсом: слева — таблица точек наблюдения, справа — карта Иркутска на Leaflet
- Каждая точка имеет: координаты, категорию грунта (I–IV по СП 14.13330), описание, данные скважины (глубина, слои)
- Форма добавления точки наблюдения: координаты (кликом на карте или вводом), категория грунта, добавление слоёв разреза (название, мощность, Vs, Vp, плотность)
- Таблица слоёв грунтового разреза отображается как стратиграфическая колонка с цветовой кодировкой по типу
- При клике на объект инфраструктуры на карте — отображается грунтовой разрез в этой точке и категория сейсмичности
- Кнопка экспорта грунтовых данных в CSV для использования в расчётных программах

## Out of scope
- Расчёты усиления на основе грунтов (отдельная задача)
- Интеграция с внешними геологическими базами данных

## Steps
1. **Таблицы грунтов в схеме** — `soil_profiles` (точка, категория по СП, описание, привязка к объекту) и `soil_layers` (id профиля, порядковый номер, название слоя, мощность в м, Vs м/с, Vp м/с, плотность кг/м³, тип).
2. **API грунтов** — CRUD endpoints для `soil_profiles` и `soil_layers`; запрос "профиль для точки" по координатам или по id объекта.
3. **Страница "Грунты"** — двухпанельный layout: таблица точек + Leaflet-карта Иркутска (center: [52.29, 104.30]) с маркерами, цвет которых зависит от категории грунта.
4. **Форма точки наблюдения** — выбор координат кликом на карте, выпадающий список категории (I — скала, II — средние, III — мягкие, IV — слабые), редактируемая таблица слоёв.
5. **Стратиграфическая колонка** — компонент визуализации разреза: вертикальная шкала глубины, прямоугольники слоёв с цветом по типу, подписи Vs.
6. **Предзаполненные данные** — внести 5–8 реальных точек наблюдения Иркутска с известными грунтовыми условиями (центр, Свердловский р-н, Ленинский р-н, набережная).
7. **Экспорт в CSV** — кнопка выгрузки профиля выбранной точки.
