# 🌐 SeismoNet Иркутск

### Сеть сейсмических наблюдений за объектами гражданской и промышленной инфраструктуры г. Иркутска

---

## О проекте https://seismonet.ru

**SeismoNet** — профессиональная система мониторинга сейсмической безопасности строительных объектов Иркутска. Система предназначена для организаций, осуществляющих строительство и эксплуатацию гражданских и промышленных объектов в сейсмоопасном регионе.

Платформа объединяет:
- сеть датчиков (акселерометров) на объектах
- онлайн-концентратор сейсмических данных
- инструменты инженерного анализа (грунты, усиление, спектры)
- нормативную базу (СП, СНиП, ГОСТ)
- автоматические оповещения при сейсмических событиях

## Возможности

### 🏗 Объекты мониторинга
- Реестр инфраструктурных объектов с адресами, категориями и характеристиками
- Привязка застройщиков к объектам
- Геологические профили по каждому объекту (скважины, литология, категория грунта по СП 14.13330.2018)
- Интерактивная карта объектов (Leaflet)

### 📡 Датчики и станции
- Управление сетью сейсмических станций
- Мониторинг статуса (онлайн / деградация / отключена)
- Контроль заряда батарей, объёма памяти, качества данных
- Онлайн-концентратор: живые осциллограммы с каждого канала в реальном времени (WebSocket)

### 📊 Анализ данных
- **Сейсмограммы** — просмотр записей, каталог событий
- **Спектральный анализ** — Фурье-спектры, спектры отклика, ANSA
- **Инженерная геология** — база грунтовых профилей с параметрами Vs30, f₀, коэффициент усиления
- **История расчётов** — архив всех выполненных расчётов с заметками и версионированием

### 🔬 Расчёты и проектирование
- Метод тонких слоёв (МТСМ) — расчёт усиления грунта
- МКЭ (метод конечных элементов) — моделирование отклика конструкций
- Спектры ответа конструкций, спектры ANSA
- Генерация PDF-отчётов (с кириллицей)

### 📚 Нормативная база
- Актуальные нормы: СП 14.13330.2022, СНиП II-7-81*, ГОСТ 17516.1-90 и другие
- Карты ОСР-2015 (A, B, C)
- Таблицы сейсмических категорий грунтов

### 🔔 Оповещения
- Автоматические уведомления при превышении пороговых значений
- Интеграция: **Telegram**, **Email (Unisender)**, **Slack**
- Синхронизация с USGS и JMA (Японское метеорологическое агентство)

## Архитектура

```
client/          — React-приложение
  src/
    pages/       — страницы (HomePage, Analysis, Calculations, ...)
    components/  — компоненты (layout, ui, maps, charts, ...)
    hooks/       — хуки (useAuth, use-toast, ...)
    lib/         — утилиты (queryClient, ...)

server/          — Express-сервер
  index.ts       — точка входа, инициализация
  routes.ts      — REST API маршруты
  storage.ts     — интерфейс к базе данных
  auth.ts        — аутентификация

shared/
  schema.ts      — Drizzle-схема БД + Zod-типы (общие для клиента и сервера)
```

## Быстрый старт

### Требования
- Node.js 20+
- PostgreSQL (или Neon serverless)

### Установка

```bash
git clone https://github.com/tulubyev/SeismoNet.git
cd SeismoNet
npm install
```

### Переменные окружения

Создайте файл `.env`:

```env
DATABASE_URL=postgresql://user:password@host:5432/dbname
SESSION_SECRET=your-secret-key
TELEGRAM_BOT_TOKEN=...       # опционально
TELEGRAM_CHAT_ID=...         # опционально
UNISENDER_API_KEY=...        # опционально
SLACK_BOT_TOKEN=...          # опционально
```

### Инициализация базы данных

```bash
npm run db:push
```

### Запуск в режиме разработки

```bash
npm run dev
```

Приложение будет доступно по адресу: `http://localhost:5000`

### Продакшн-сборка

```bash
npm run build
npm start
```
